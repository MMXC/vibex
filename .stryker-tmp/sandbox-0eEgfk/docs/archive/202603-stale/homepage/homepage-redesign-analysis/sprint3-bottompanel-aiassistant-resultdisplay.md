### 3.3 Story 7.2 - 智能诊断 (4h, P1)

**核心行为**:
- 自动分析需求，指出潜在问题
- 问题卡片: 标题 + 严重程度 + 建议
- 快速修复: 一键应用到输入框

**API 调用**:
```typescript
// POST /api/v1/analysis/diagnose
interface DiagnoseRequest {
  requirement: string;
}
interface DiagnoseResult {
  issues: Array<{
    id: string;
    title: string;        // 如: "需求模糊"
    severity: 'error' | 'warning' | 'info';
    suggestion: string;   // 如: "建议补充具体场景"
    quickFix?: string;   // 可选快速修复文本
  }>;
}
```

**UI 组件**:
```tsx
<DiagnosisPanel>
  {issues.map(issue => (
    <IssueCard severity={issue.severity}>
      <IssueTitle>{issue.title}</IssueTitle>
      <IssueSuggestion>{issue.suggestion}</IssueSuggestion>
      {issue.quickFix && (
        <QuickFixButton onApply={() => applyFix(issue.quickFix)}>
          快速修复
        </QuickFixButton>
      )}
    </IssueCard>
  ))}
</DiagnosisPanel>
```
