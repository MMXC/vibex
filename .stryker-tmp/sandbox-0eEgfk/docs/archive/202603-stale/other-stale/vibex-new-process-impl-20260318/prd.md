## 1. 项目概述

基于 VibeX 新流程设计技术指导文档，实现5步流程架构：

| 步骤 | 名称 | 用户操作 | 产出 |
|------|------|----------|------|
| 1 | 需求录入 | 输入产品需求描述 | 需求文本 |
| 2 | 限界上下文 | AI识别+用户勾选确认 | 选定的限界上下文 |
| 3 | 业务流程 | AI生成+用户调整 | 业务流程方案 |
| 4 | UI组件 | AI生成+用户选择 | UI组件方案 |
| 5 | 创建项目 | 汇总信息 | 完整项目 |

## 2. 成功指标

| 指标 | 目标 |
|------|------|
| 用户可在首页启动5步流程 | 5个步骤入口可见 |
| 单步加载时间 | ≤ 2s |
| 整体流程完成时间 | ≤ 30s |
| 数据持久化 | 刷新页面不丢失 |
