# Spec: S1.1 新增独立流程入口

## 6.1 Store 层修改

### 6.1.1 `addFlowNode` 签名变更

**文件**: `src/lib/canvas/canvasStore.ts`

**当前**:
```typescript
addFlowNode: (data: BusinessFlowDraft) => void;
```

**变更后**:
```typescript
addFlowNode: (data: BusinessFlowDraft) => void;
// 支持 contextId 可选，为 null 时创建"独立流程"
```

### 6.1.2 新增 action: `addStandaloneFlow`

如需区分独立流程与上下文绑定流程，可新增：

```typescript
addStandaloneFlow: (name?: string) => void;
```

实现：
```typescript
addStandaloneFlow: (name) => {
  const newNode: BusinessFlowNode = {
    nodeId: generateId(),
    contextId: '', // 空字符串表示独立流程
    name: name || `新建流程 ${get().flowNodes.length + 1}`,
    steps: [],
    confirmed: false,
    status: 'pending',
    children: [],
  };
  set((s) => ({ flowNodes: [...s.flowNodes, newNode] }));
}
```

## 6.2 UI 层修改

### 6.2.1 移除 `canManualAdd` 条件

**文件**: `src/components/canvas/BusinessFlowTree.tsx`

**当前** (约第 370 行):
```typescript
const canManualAdd = contextNodes.length > 0;

const handleManualAdd = useCallback(() => {
  // ...
}, [contextNodes, addFlowNode]);
```

**变更后**:
```typescript
// 移除 canManualAdd，直接允许添加
const handleManualAdd = useCallback(() => {
  const unconfirmedCtx = contextNodes.find((c) => !c.confirmed);
  const targetCtx = unconfirmedCtx ?? contextNodes[0];
  if (targetCtx) {
    addFlowNode({ contextId: targetCtx.nodeId, name: `${targetCtx.name}业务流程`, steps: [] });
  } else {
    // 独立流程：contextId 为空
    const count = useCanvasStore.getState().flowNodes.length;
    addFlowNode({ contextId: '', name: `新建流程 ${count + 1}`, steps: [] });
  }
}, [contextNodes, addFlowNode]);
```

### 6.2.2 工具栏按钮始终显示

**当前**:
```tsx
{canManualAdd && (
  <button className={styles.btnAddFlow} onClick={handleManualAdd}>
    + 添加流程
  </button>
)}
```

**变更后**:
```tsx
<button className={styles.btnAddFlow} onClick={handleManualAdd}>
  + 添加流程
</button>
```

### 6.2.3 新增后自动展开

在 `handleManualAdd` 成功后，需触发新增节点的展开状态。可通过 store 维护 `expandedFlowIds` Set 实现，或在 UI 层做即时展开。

describe('addFlowNode - S1.1', () => {
  it('S1.1-AC4: 零上下文时可新增流程', () => {
    useCanvasStore.setState({ contextNodes: [], flowNodes: [] });
    const { addFlowNode } = useCanvasStore.getState();
    addFlowNode({ contextId: '', name: '独立流程', steps: [] });
