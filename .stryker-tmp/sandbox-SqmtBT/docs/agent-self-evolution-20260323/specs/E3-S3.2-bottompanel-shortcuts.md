# Spec: E3-S3.2 — BottomPanel 快捷功能实现

**Epic**: Epic 3 首页交互功能补全  
**Story**: BottomPanel 快捷功能实现  
**Agent**: dev  
**状态**: Ready

---

## 概述

实现首页 BottomPanel 的 4 个快捷功能，使快捷键和快捷按钮有实际功能。

---

## 功能描述

### 快捷功能列表
1. **Ctrl+S / 保存** — 保存当前分析进度
2. **Ctrl+Z / 撤销** — 撤销上一步操作
3. **Ctrl+Shift+Z / 重做** — 重做已撤销的操作
4. **Ctrl+P / 预览** — 切换预览模式

### 实现要求
- 键盘快捷键与按钮功能一一对应
- 快捷键冲突检测（与其他应用/浏览器默认行为）
- 状态反馈（已保存/未保存）
- 操作历史栈管理（撤销/重做）
