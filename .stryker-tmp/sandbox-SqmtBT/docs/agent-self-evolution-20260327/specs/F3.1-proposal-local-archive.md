# Spec: F3.1 — 提案文件本地归档

## 功能概述
每日自检提案保存到 `proposals/YYYYMMDD/{agent}.md`，心跳脚本增加目录存在性检查，确保提案可靠归档。

## 实现方案

### 提案目录结构
```
vibex/proposals/
├── 20260327/
│   ├── analyst.md
│   ├── pm.md
│   ├── dev.md
│   ├── architect.md
│   ├── tester.md
│   └── reviewer.md
└── 20260326/
    └── ...
```

### 心跳脚本增加归档检查
```bash
# 创建提案目录
PROPOSAL_DIR="$VIBEX_DIR/proposals/$(date +%Y%m%d)"
mkdir -p "$PROPOSAL_DIR"

# 写入提案文件
PROPOSAL_FILE="$PROPOSAL_DIR/pm.md"
cat > "$PROPOSAL_FILE" << 'EOF'
# PM 每日自检提案

## 本次自检摘要
...
EOF

echo "✅ 提案已归档: $PROPOSAL_FILE"
```
