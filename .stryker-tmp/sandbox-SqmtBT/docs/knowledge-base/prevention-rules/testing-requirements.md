# 测试要求

## 单元测试要求

### 必须测试的场景

1. **边界条件**
   - 空数组/对象
   - undefined/null 值
   - 极端值

2. **错误处理**
   - API 错误
   - 网络错误
   - 类型错误

3. **状态管理**
   - 初始状态
   - 状态更新
   - 状态重置
