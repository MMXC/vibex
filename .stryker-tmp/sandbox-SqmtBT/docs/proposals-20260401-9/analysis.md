**来源**: `vibex-canvas-checkbox-unify/analysis.md`（2026-03-30）


### 3.3 E3: 响应式布局与移动端体验 — P1

**来源**: Sprint 1 复盘会议 IM-4 + UX 优化目标

**问题**:
- Canvas 三列布局在大屏幕上正常，但平板/手机端不可用
- 卡片在小屏幕上溢出，滚动体验差
- 抽屉菜单在小屏幕上遮挡内容

**解决方案**:

| 断点 | 布局调整 |
|------|---------|
| ≥1280px | 三列（树 + 画布 + 抽屉） |
| 768-1279px | 两列（树 + 画布），抽屉 overlay |
| <768px | 单列（Tab 切换树/画布/抽屉） |

**工时估算**: 5-7h

**验收标准**:
```
- 768px 断点下画布可正常操作
- 移动端 Tab 切换三视图可用
- 抽屉 overlay 不遮挡画布主体
```
