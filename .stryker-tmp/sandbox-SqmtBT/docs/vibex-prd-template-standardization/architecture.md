# Architecture: vibex-prd-template-standardization

**Project**: PRD 模板标准化
**Agent**: coord (代写)
**Date**: 2026-03-31

---

## 1. 执行摘要

制定强制统一的 PRD 模板，所有提案项目必须使用，减少驳回率。

## 2. 模板规范

PRD 必须包含：
- 执行摘要（背景/目标/成功指标）
- Epic 拆分表（ID/描述/工时/验收标准）
- 验收标准总表（ID/条件/测试断言）
- 实施计划表（Epic/Story/工时/负责人）

## 3. 落地方式

在 phase1 模板中强制嵌入 PRD 模板，检查不符合则拒绝进入 design-architecture
