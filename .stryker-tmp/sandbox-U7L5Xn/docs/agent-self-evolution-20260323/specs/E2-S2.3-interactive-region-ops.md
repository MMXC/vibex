# Spec: E2-S2.3 — 交互式区域操作

**Epic**: Epic 2 ReactFlow 可视化能力整合  
**Story**: 交互式区域操作  
**Agent**: dev  
**Status**: Ready

---

## 概述

支持用户在可视化画布上选择局部区域，对选定区域触发重新生成操作。

---

## 功能描述

### 区域选择
- 框选工具: 在画布上拖拽创建选择框
- 多选: 支持 Ctrl/Cmd + 点击多选
- 全选: 支持 Ctrl/Cmd + A 全选
- 反选: 右键菜单支持反选

### 区域重生成
- 触发方式: 选中区域后点击"重新生成"按钮
- 确认流程: 弹窗确认（显示将重新生成的内容摘要）
- 执行: 调用 `/ddd/regenerate` 端点
- 结果: 更新选中区域的节点，保留其他区域不变

---

## 验收标准

| ID | Given | When | Then |
|----|-------|------|------|
| AC1 | 框选 | 拖拽创建选择框 | 选中区域高亮显示 |
| AC2 | 多选 | Ctrl+点击 | 多个节点被选中 |
| AC3 | 重生成触发 | 点击按钮 | 弹窗显示确认信息 |
| AC4 | 确认执行 | 点击确认 | API 调用，更新节点 |
| AC5 | 取消执行 | 点击取消 | 区域保持不变 |
| AC6 | 未选中 | 点击重生成 | 提示"请先选择区域" |
