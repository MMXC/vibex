# Spec: E4-S4.1 — team-tasks 提案字段扩展

**Epic**: Epic 4 提案生命周期管理  
**Story**: team-tasks 提案字段扩展  
**Agent**: coord  
**状态**: Ready

---

## 概述

在 team-tasks 系统的任务 JSON 中增加 `proposal-origin` 字段，记录任务来源。

---

## 功能描述

### 字段设计
```json
{
  "stages": {
    "task-id": {
      "proposalOrigin": {
        "agent": "pm",
        "proposalFile": "proposals/20260323/pm-self-check.md",
        "proposalId": "P1",
        "submittedAt": "2026-03-23T05:00:00Z"
      }
    }
  }
}
```

### 使用场景
- 创建新任务时指定来源
- 查询任务时展示来源信息
- 统计各 Agent 提案落地率

---

## 验收标准

| ID | Given | When | Then |
|----|-------|------|------|
| AC1 | 新建任务 | 指定 proposal-origin | 字段正确写入 JSON |
| AC2 | 查询任务 | 不指定字段 | 不影响现有逻辑（向后兼容） |
| AC3 | 无来源任务 | 查询来源 | 字段为 null/undefined |
| AC4 | 字段验证 | 无效值 | 拒绝写入，记录错误 |
