## 1. 概述

### 1.1 背景

当前首页点击"开始设计"后跳转到 `/confirm`，用户体验不连贯。需重构为单页式五步流程。

### 1.2 问题

| 问题 | 影响 |
|------|------|
| 页面跳转 | 体验不连贯 |
| 无预览区域 | 无法实时看结果 |
| AI 助手固定 | 占用空间 |
