## 1. 问题描述

当前首页实现不符合产品设计要求，需要重新设计实现。

---

## 2. 设计要求

### 2.1 整体布局
- **Hero 区域**: 单行居中主副标题形式（水平居中）
- **整体结构**: 
  - 顶部：产品一句话功能说明（居中）
  - 中部：预览区域（实时渲染图形）
  - 底部：需求输入框

### 2.2 单页式生成流程

**左侧：步骤导航**
- 显示步骤 1-4（需求输入 → 上下文分析 → 模型设计 → 流程设计 → UI 生成）
- 当前步骤高亮
- 可点击切换步骤
- 支持撤销操作

**中间：实时预览区域**
- Mermaid 实时渲染：
  - 上下文图 (Context Diagram)
  - 模型图 (Domain Model)
  - 流程图 (Business Flow)
  - UI 节点图 (UI Wireframe)
- 用户输入需求时实时更新预览

**右侧/底部：编辑区域**
- 当前步骤的编辑表单
- 支持调整和修改

### 2.3 功能要求

| 功能 | 描述 |
|------|------|
| Hero 居中 | 主副标题水平居中显示 |
| 状态保持 | 登录后保留用户输入内容 |
| 模板可用 | 点击模板有响应，填入需求 |
| 步骤导航 | 左侧显示步骤，支持切换 |
| 实时预览 | Mermaid 实时渲染各类图表 |
| 单页流程 | 首页完成全流程，无需跳转 |
