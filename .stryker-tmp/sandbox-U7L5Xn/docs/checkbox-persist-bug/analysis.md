# Checkbox 勾选状态持久化 Bug 分析

## 问题描述

**来源**: 小羊反馈 (2026-04-02)

**现状问题**:
1. 组件树点击后请求体包含未勾选节点
2. checkbox 勾选字段未持久化
3. 上下文树/卡片树/组件树的勾选状态没有保存到数据中

**期望行为**:
- 勾选字段应持久化到 JSON 数据中
- 不应仅作为状态管理
- 请求后根据 JSON 里面该卡片的勾选字段值构造 prompt
- 一键导入时勾选状态也可导入

## 涉及文件

- BoundedContextTree.tsx (上下文树)
- ComponentTree.tsx (卡片树/组件树)
- BusinessFlowTree.tsx (流程树)

## 解决方案

1. 在数据结构中增加 `selected` 或 `confirmed` 字段
2. 勾选时更新该字段到 JSON 数据
3. 取消勾选时也更新该字段
4. 请求时从 JSON 数据读取勾选状态构造 prompt

## 验收标准

- [ ] 上下文树卡片勾选后持久化
- [ ] 卡片树勾选后持久化
- [ ] 组件树勾选后持久化
- [ ] 请求时根据持久化字段构造 prompt
- [ ] 一键导入时勾选状态正确导入
