# 代码审查清单

> 代码审查时必须检查的项目

---

## 必查项目

### 代码质量

- [ ] 无 `any` 类型
- [ ] 变量/函数有明确类型
- [ ] 错误处理完善
- [ ] 无硬编码值

### 安全

- [ ] 无敏感信息泄露
- [ ] 输入验证
- [ ] XSS 防护

### 性能

- [ ] 无不必要的重新渲染
- [ ] 懒加载适当使用
- [ ] 无内存泄漏

### 测试

- [ ] 关键逻辑有测试
- [ ] 测试覆盖达标

---

## React 特定

- [ ] 组件 props 定义完整
- [ ] useEffect 依赖完整
- [ ] 状态初始化正确
- [ ] 无内存泄漏

---

## 提交前自检

```bash
# 运行测试
npm test

# 运行类型检查
npx tsc --noEmit

# 运行 lint
npm run lint
```
