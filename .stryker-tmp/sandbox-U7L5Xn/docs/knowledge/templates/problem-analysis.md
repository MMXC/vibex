# Template: 问题分析模板 (Problem Analysis Template)

## 使用场景
当发现一个新的 bug 或问题时，使用此模板进行结构化分析。

---

## 1. 问题概述

| 字段 | 内容 |
|------|------|
| **问题ID** | `prob-YYYYMMDD-NNN` |
| **标题** | 简明描述问题本质 |
| **严重程度** | P0 / P1 / P2 / P3 |
| **发现时间** | YYYY-MM-DD |
| **发现人** | @username |

> 描述当前实际发生的行为。
