## Dev 红线（禁止项）

1. **禁止 `any` 类型**: 所有函数参数和返回值必须有明确类型，违者 CI 失败
2. **禁止直接操作 DOM**: 必须通过 React 组件，禁止 `document.getElementById`
3. **禁止同步 localStorage 阻塞渲染**: 读取必须为异步或在 `useEffect` 中
4. **禁止 CSS 硬编码颜色**: 必须使用 CSS Variables (`--theme-*`)，禁止 `#fff` 等硬编码
5. **禁止修改已有组件样式**: ThemeContext 只做状态管理，不改其他组件

### CSS Variables 规范

```css
/* theme-variables.css */
:root {
  --theme-bg-primary: var(--bg, #ffffff);
  --theme-text-primary: var(--text, #111827);
  --theme-border: var(--border, #e5e7eb);
  /* 亮色变量 */
  --light-bg: #ffffff;
  --light-text: #111827;
  --light-border: #e5e7eb;
  /* 暗色变量 */
  --dark-bg: #0a0a0f;
  --dark-text: #f9fafb;
  --dark-border: #374151;
}

[data-theme="light"] {
  --theme-bg-primary: var(--light-bg);
  --theme-text-primary: var(--light-text);
  --theme-border: var(--light-border);
}

[data-theme="dark"] {
  --theme-bg-primary: var(--dark-bg);
  --theme-text-primary: var(--dark-text);
  --theme-border: var(--dark-border);
}
```
