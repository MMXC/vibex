## 概述

tester Agent 优先验证 vibex-simplified-flow 的 `/ddd/business-domain` API 端点，确保核心流程可用。

---

## 功能描述

### 验证目标
- 端点: `POST /api/ddd/business-domain`
- 预期: 返回 200，响应包含 `domainModel` 字段

### 验证步骤
1. 构造测试请求（包含必要参数）
2. 调用端点
3. 验证响应状态码
4. 验证响应数据结构
5. 报告验证结果

---

## 验收标准

| ID | Given | When | Then |
|----|-------|------|------|
| AC1 | 端点正常 | 调用 | expect(response.status).toBe(200) |
| AC2 | 响应有效 | 返回 200 | expect(response.data.domainModel).toBeDefined() |
| AC3 | 端点异常 | 返回 4xx/5xx | expect.fail，记录错误，上报 #coord |
| AC4 | 请求超时 | 5s 内无响应 | expect.fail，记录超时，上报 #coord |
