# Spec: E2-S2.1 — JSON 树可视化

**Epic**: Epic 2 ReactFlow 可视化能力整合  
**Story**: JSON 树节点展示  
**Agent**: dev  
**状态**: Ready

---

## 概述

在 DDD 建模页面实现 JSON 树可视化展示，将领域模型数据渲染为可交互的节点树。

---

## 功能描述

### 输入
- DDD 建模输出的领域模型 JSON

### 展示要求
- 节点类型: DomainModel, Entity, ValueObject, Aggregate, Service, Repository, Event
- 节点显示: 名称 + 类型标签 + 展开/收起状态
- 层级: 支持 ≥ 3 层嵌套展示
- 节点数量: 支持 ≥ 100 个节点无性能问题

### 交互能力
- 点击节点: 展开/收起子节点
- 悬停节点: 显示详情浮层
- 双击节点: 打开编辑面板

---

## 验收标准

| ID | Given | When | Then |
|----|-------|------|------|
| AC1 | 领域模型数据 | 页面加载 | expect(treeNodes.length > 0) |
| AC2 | 展开节点 | 点击 | 子节点可见 |
| AC3 | 收起节点 | 再次点击 | 子节点隐藏 |
| AC4 | 详情浮层 | 悬停 | 200ms 内显示 |
| AC5 | ≥100 节点 | 渲染 | 首次渲染 < 1s |
