# Spec: E4-S4.2 — 提案状态查询能力

**Epic**: Epic 4 提案生命周期管理  
**Story**: 提案状态查询能力  
**Agent**: coord  
**状态**: Ready

---

## 概述

提供提案落地状态的查询能力，支持按状态筛选和详情查看。

---

## 功能描述

### 查询接口
- CLI: `task_manager.py query --proposal-origin <agent> --status <status>`
- 支持筛选: agent, status, date-range

### 状态定义
| 状态 | 描述 |
|------|------|
| pending | 提案待评审 |
| approved | 提案已批准，待开发 |
| in-progress | 开发中 |
| completed | 已完成验收 |
| rejected | 提案被驳回 |

---

## 验收标准

| ID | Given | When | Then |
|----|-------|------|------|
| AC1 | 查询全部 | 不带参数 | 返回所有提案 |
| AC2 | 按 agent 筛选 | --agent pm | 仅返回 pm 的提案 |
| AC3 | 按状态筛选 | --status pending | 仅返回 pending 的提案 |
| AC4 | 组合筛选 | --agent pm --status approved | 正确交集 |
| AC5 | 空结果 | 筛选无匹配 | 提示"无结果" |
