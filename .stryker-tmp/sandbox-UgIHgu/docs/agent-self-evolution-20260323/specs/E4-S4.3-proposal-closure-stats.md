# Spec: E4-S4.3 — 提案落地率统计

**Epic**: Epic 4 提案生命周期管理  
**Story**: 提案落地率统计  
**Agent**: coord  
**状态**: Ready

---

## 概述

生成提案落地率统计报告，支持按 Agent、时间范围等维度统计。

---

## 功能描述

### 统计指标
- 各 Agent 提案数量
- 各状态分布
- 落地率 = completed / (completed + rejected) * 100%
- 平均处理时长

### 报告格式
```markdown
## 提案落地率统计
**时间范围**: 2026-03-01 ~ 2026-03-23

| Agent | 提案数 | 完成 | 驳回 | 落地率 |
|-------|--------|------|------|--------|
| pm | 5 | 3 | 1 | 75% |
| dev | 8 | 5 | 2 | 71% |
| ...

**总体落地率**: 68%
**平均处理时长**: 3.2 天
```

## 验收标准

| ID | Given | When | Then |
|----|-------|------|------|
| AC1 | 统计执行 | 有提案数据 | 生成报告 |
| AC2 | 无提案数据 | 统计执行 | 报告显示"无数据" |
| AC3 | 按时间筛选 | 指定范围 | 仅统计范围内提案 |
| AC4 | 落地率计算 | 有完成/驳回 | 计算正确 |
| AC5 | 平均时长计算 | 有处理记录 | 计算正确 |
