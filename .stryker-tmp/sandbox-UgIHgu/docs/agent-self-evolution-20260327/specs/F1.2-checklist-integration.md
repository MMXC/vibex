# Spec: F1.2 — 经验沉淀检查清单集成

## 功能概述
在各 Agent 的任务收尾流程中增加经验沉淀强制检查步骤，确保经验条目不会被遗忘。

## 执行步骤

1. **修改各 Agent 的 HEARTBEAT.md**
   - 在 HEARTBEAT.md 模板底部「经验沉淀索引」表格上方增加检查清单区
   - 格式：
   ```markdown
   ## 任务收尾经验沉淀 Checklist
   
   - [ ] 识别本次任务的经验教训（至少 1 条）
   - [ ] 写入 HEARTBEAT.md「经验沉淀索引」表格
   - [ ] 提案归档到 `proposals/YYYYMMDD/{agent}.md`
   ```

2. **修改 SOUL.md / AGENTS.md**
   - 在「任务完成」流程中增加经验沉淀步骤
   - 强调：经验沉淀是 DoD 的一部分，未完成不得标记 HEARTBEAT_OK

## DoD
- [ ] HEARTBEAT.md 包含「任务收尾经验沉淀 Checklist」章节
- [ ] Checklist 包含 3 项必要步骤
- [ ] 格式规范，可在 Markdown 中渲染为 TODO
