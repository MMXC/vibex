# Spec: F2.1 — Feishu 群组 ID 配置修复

## 功能概述
各 Agent 的 HEARTBEAT.md 中 Feishu 群组 ID 应统一为正确值 `oc_8bf15f971d009f363f333d37c3bede2e`，替换错误值 `C0ANZ3J40LT`。

## 执行步骤

1. **扫描所有 HEARTBEAT.md**
   ```bash
   grep -r "C0ANZ3J40LT\|oc_495e9c\|target" \
     /root/.openclaw/workspace-*/HEARTBEAT.md \
     /root/.openclaw/vibex/workspace-*/HEARTBEAT.md \
     2>/dev/null
   ```

2. **确认正确 ID**
   - oc_8bf15f971d009f363f333d37c3bede2e (主要通知群)
   - oc_13a5179e7695001c5a037b41e81dfa75 (coord 群)

3. **逐个修复**
   - 对每个 HEARTBEAT.md，用正确 ID 替换错误 ID
   - 修复后用 grep 验证
