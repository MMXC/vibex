# Spec: E2-S2.2 — Mermaid 画布集成

**Epic**: Epic 2 ReactFlow 可视化能力整合  
**Story**: Mermaid 画布集成  
**Agent**: dev  
**状态**: Ready

---

## 概述

将 Mermaid 图表渲染为 ReactFlow 自定义节点，支持在画布上展示和交互 DDD 每步的 Mermaid 图表。

---

## 功能描述

### 输入
- DDD 每步生成的 Mermaid 图表定义

### 渲染要求
- Mermaid → ReactFlow 节点映射
- 支持节点类型: flowchart, sequence, class, state
- 节点样式与整体 ReactFlow 风格一致

### 交互能力
- 点击节点: 展开为完整图表
- 节点内缩放: 支持放大/缩小
- 节点连接: 支持拖拽连接线

---

## 验收标准

| ID | Given | When | Then |
|----|-------|------|------|
| AC1 | Mermaid 定义 | 渲染 | expect(mermaidNodes.length > 0) |
| AC2 | 展开操作 | 点击节点 | 完整图表显示 |
| AC3 | 缩放操作 | 滚轮/按钮 | 图表缩放正常 |
| AC4 | 连接操作 | 拖拽 | 连接线生成 |
| AC5 | 节点溢出 | 图表过大 | 自动裁剪 + 滚动 |
