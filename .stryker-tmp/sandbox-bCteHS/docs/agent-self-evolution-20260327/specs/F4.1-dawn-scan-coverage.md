# Spec: F4.1 — 凌晨时段心跳覆盖

## 功能概述
将 PM 心跳扫描扩展到 0-6 点时段（当前主要覆盖 8-23 点），确保长任务不遗漏。

## 当前状态

```
# 当前 PM cron 配置 (示例)
0 8,10,12,14,16,18,20,22 * * * bash /root/.openclaw/scripts/heartbeats/pm-heartbeat.sh
```

缺少: 0,1,2,3,4,5,6,7 点

## 目标配置

```
# 目标 PM cron 配置
0 */2 * * * bash /root/.openclaw/scripts/heartbeats/pm-heartbeat.sh
```
每 2 小时执行一次，覆盖全天 0-23 点。

## 验收标准

```javascript
// test-f4.1.js
const { execSync } = require('child_process');
const crontab = execSync('crontab -l', { encoding: 'utf8' });
const pmLine = crontab.split('\n').find(l => l.includes('pm-heartbeat'));
expect(pmLine).toBeTruthy();

// 验证覆盖 0-6 点
expect(pmLine).toMatch(/pm-heartbeat.*\*\/2|0.*pm-heartbeat.*1.*pm-heartbeat.*2|pm-heartbeat.*0|pm-heartbeat.*1|pm-heartbeat.*2|pm-heartbeat.*3|pm-heartbeat.*4|pm-heartbeat.*5|pm-heartbeat.*6/);
```

```bash
# 验证 cron 配置
crontab -l | grep pm-heartbeat

# 验证心跳间隔 ≤ 2 小时

```
