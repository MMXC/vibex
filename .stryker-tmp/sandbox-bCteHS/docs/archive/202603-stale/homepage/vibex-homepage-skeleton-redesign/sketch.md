# 骨架屏布局草图 - 请确认

## 1. 整体布局结构

```
┌─────────────────────────────────────────────────────────────┐
│                      HEADER (固定)                        │
│         Logo                              Nav Links        │
├────────────┬─────────────────────────────┬────────────────┤
│            │                             │                │
│  SIDEBAR  │      主工作区 (可变)       │  右侧对话     │
│  (固定)    │                             │   (固定)      │
│            │  ┌─────────────────────┐  │                │
│  - 流程   │  │   预览区域 (60%)   │  │  - AI 对话   │
│  - 项目   │  │   (可拖拽/互换)    │  │  - 思考过程  │
│            │  └─────────────────────┘  │                │
│            │  ┌─────────────────────┐  │                │
│            │  │   录入区域 (40%)   │  │                │
│            │  │   (可拖拽/互换)    │  │                │
│            │  └─────────────────────┘  │                │
├────────────┴─────────────────────────────┴────────────────┤
│                    底部操作栏 (固定)                      │
│              [上一步] [开始设计] [下一步]                   │
└─────────────────────────────────────────────────────────────┘
```

---

## 2. 固定区域 vs 可变区域

### 固定区域 (不可变)
| 区域 | 说明 |
|------|------|
| Header | Logo + 导航链接 |
| Sidebar | 流程进度 + 项目结构 |
| 右侧对话 | AI 思考过程 |
| 底部操作栏 | 步骤按钮 |

### 可变区域 (可调整)
| 区域 | 说明 |
|------|------|
| 预览区域 | 大小可拖拽调整，可与录入区域互换位置 |
| 录入区域 | 大小可拖拽调整，可与预览区域互换位置 |

---

## 3. 拖拽交互说明

### 调整大小
- 拖拽预览/录入区域分隔线
- 调整两侧高度比例

### 互换位置
- 拖拽区域到对方位置释放
- 自动切换上下位置
