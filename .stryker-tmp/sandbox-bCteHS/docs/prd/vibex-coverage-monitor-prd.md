## 3. 告警规则

| 阈值类型 | 默认值 | 告警级别 |
|----------|--------|----------|
| 语句覆盖率 | < 60% | 🔴 阻断 |

## 6. 技术方案

### 6.1 配置

```yaml
# .github/workflows/coverage.yml
name: Coverage Monitor

on:
  pull_request:
    paths:
      - 'src/**'
      - 'tests/**'

jobs:
  coverage:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Run tests with coverage
        run: npm test -- --coverage --coverageThreshold='{"global":{"statements":60,"branches":50,"functions":60}}'
      
      - name: Check coverage
        run: |
          npx coverage-threshold ...
```

### 6.2 覆盖率报告格式

```json
{
  "timestamp": "2026-03-04T10:00:00Z",
  "total": {
    "statements": 65.5,
    "branches": 52.3,
    "functions": 68.1,
    "lines": 64.8
  },
  "files": [
    {
      "path": "src/app/chat/page.tsx",
      "statements": 25.0,
      "branches": 20.0,
      "functions": 30.0,
      "status": "warning"
    }
  ],
  "threshold": {
    "statements": 60,
    "branches": 50,
    "functions": 60
  }
}
```
