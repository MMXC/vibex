# Spec: S1.3 流程节点样式标准化

## 6.1 现状分析

### 两套节点样式体系

| 视图 | 文件 | 主题 | 使用场景 |
|------|------|------|----------|
| 树形列表 | `canvas.module.css` | 浅色（白底+彩色边框） | `BusinessFlowTree` |
| 画布图 | `FlowNodes.tsx` + `FlowNodes.module.css` | 深色（深灰底+渐变边框） | React Flow 画布 |

**设计决策**: 两套样式服务于不同视图，**不合并**，但各自需符合设计规范。

### 6.2.3 统一规范

```css
/* === 状态颜色常量 === */
:root {
  --status-pending-color: #f59e0b;   /* amber-500 */
  --status-confirmed-color: #10b981; /* emerald-500 */
  --status-error-color: #ef4444;     /* red-500 */
  --status-border-width: 2px;
}

/* === Flow Card === */
.flowCard {
  border-radius: 8px;
  border: var(--status-border-width) solid transparent;
  background: #ffffff;
  transition: box-shadow 0.2s;
}

.flowCard.nodePending {
  border-color: var(--status-pending-color);
  background: #fffbeb; /* amber-50 */
}

.flowCard.nodeConfirmed {
  border-color: var(--status-confirmed-color);
  background: #ecfdf5; /* emerald-50 */
}

.flowCard.nodeError {
  border-color: var(--status-error-color);
  background: #fef2f2; /* red-50 */
}

/* === Step Row === */
.stepRow {
  border-radius: 6px;
  border: var(--status-border-width) solid transparent;
  background: #f9fafb;
  padding: 8px 12px;
  margin: 4px 0;
  transition: all 0.2s;
}

.stepRow.nodePending {
  border-color: var(--status-pending-color);
  background: #fffbeb;
}

.stepRow.nodeConfirmed {
  border-color: var(--status-confirmed-color);
  background: #ecfdf5;
}

.stepRow.nodeError {
  border-color: var(--status-error-color);
  background: #fef2f2;
}
```
