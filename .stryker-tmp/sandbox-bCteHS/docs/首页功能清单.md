## 五、测试用例

### TC1: 页面加载
```
1. 打开首页
2. 预期: 页面正常显示，布局正确
```

### TC2: 步骤切换
```
1. 点击步骤2
2. 预期: currentStep=2，步骤2高亮，预览区刷新
```

### TC3: 输入发送
```
1. 输入需求文本
2. 点击发送
3. 预期: 按钮禁用，显示loading，预览区加载
```

### TC4: 底部面板拖动
```
1. 拖动收起手柄向上
2. 预期: 底部面板高度减小
3. 拖动到底部
4. 预期: 底部面板完全收起
```

### TC5: 悬浮模式
```
1. 滚动预览区
2. 预期: 底部面板缩小为悬浮栏
3. 点击收起按钮
4. 预期: 底部面板完全收起
```
