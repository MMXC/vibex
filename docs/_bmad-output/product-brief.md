# Product Brief - VibeX 前端页面原型

**项目代号**: VibeX 前端页面原型  
**状态**: Ready  
**创建时间**: 2026-02-26  
**负责人**: Analyst  

---

## 1. Problem Statement (核心痛点)

### 当前问题
- 用户需要一个具备未来科幻风格的交互式 Web 界面
- 需要区分未登录/登录状态的不同页面展示
- 缺乏统一的粒子特效和全局动效设计规范

---

## 2. Proposed Solution (解决方案)

### 2.1 页面架构

| 页面 | 条件 | 说明 |
|------|------|------|
| 首页 (Landing) | 未登录 | 顶部导航 + 落地页 + 登录横幅 |
| 控制台 (Console) | 已登录 | 左侧菜单抽屉 + 右侧 Dashboard/功能页 |

### 2.2 UI 设计规范

**风格**: FUI (Futuristic UI) - 未来科幻风格

**视觉元素**:
- 轻蓝粒子光效 (主色调: #00FFFF)
- 全息 3D 光效
- 卡片式布局 + 光流转效果
- 全局/组件级粒子特效
- 渐变过渡 + 丝滑动画

### 2.3 首页模块

1. **顶部导航栏**
   - Logo / 品牌标识
   - 导航链接

2. **落地页内容**
   - 一句话产品介绍
   - AI 对话入口 (显著 CTA)
   - 底部功能介绍卡片 (3-4 个核心功能)

3. **登录横幅** (未登录时展开)
   - 单行显示: 邮箱输入 + 验证码输入 + 登录按钮
   - 社交登录图标 (点击切换显示对应二维码)
   - 点击非登录区域自动收起

### 2.4 控制台模块

**左侧菜单** (抽屉式，可推开/收起):
- 用户驾驶舱
- 项目信息
- 模板市场
- 设置

**用户信息栏**:
- 底部固定: 用户头像 + 积分 + VIP 展示

**右侧内容区**:
- 默认展示 Dashboard
- 点击菜单项切换显示对应内容

## 4. Core User Stories (核心用户故事)

### Story 1: 未登录用户访问首页
> 作为访客，我打开首页能看到产品介绍和登录入口  
> **验收标准**: 顶部导航 + 落地页正常显示，登录横幅在顶部导航下方展开

### Story 2: 用户登录
> 作为用户，我输入邮箱/验证码完成登录  
> **验收标准**: 登录成功后页面切换至控制台

### Story 3: 已登录用户访问控制台
> 作为登录用户，我进入控制台能看到 Dashboard  
> **验收标准**: 左侧菜单抽屉可用，点击菜单项右侧内容切换

### Story 4: 社交登录
> 作为用户，我点击社交登录图标切换显示对应二维码  
> **验收标准**: 点击图标切换二维码显示，点击非登录区域收起

### Story 5: 科幻风格动效
> 作为用户，我能看到全屏粒子特效和组件级光效动画  
> **验收标准**: 粒子动画流畅，页面过渡丝滑
