# Spec: E3-S3.3 — AIPanel 核心交互实现

**Epic**: Epic 3 首页交互功能补全  
**Story**: AIPanel 核心交互实现  
**Agent**: dev  
**状态**: Ready

---

## 概述

实现首页 AIPanel 的发送和关闭功能，确保 AI 交互核心流程可用。

---

## 功能描述

### 发送功能
- 输入框: 支持多行文本
- 发送按钮: 点击或 Ctrl+Enter 触发
- 发送中: 显示 loading 状态
- 响应显示: 在面板内显示 AI 响应

### 关闭功能
- 关闭按钮: 收起 AIPanel
- 快捷键: Esc 关闭
- 确认: 有未发送内容时提示确认

### 交互细节
- 发送前: 输入验证（非空）
- 发送中: 按钮禁用，防止重复提交
- 发送失败: 显示错误，支持重试
- 响应过长: 支持滚动查看

---

## 验收标准

| ID | Given | When | Then |
|----|-------|------|------|
| AC1 | 空输入 | 点击发送 | 提示"请输入内容" |
| AC2 | 有效输入 | 点击发送 | AI 响应显示 |
| AC3 | 发送中 | 再次点击 | 按钮禁用 |
| AC4 | 发送失败 | API 错误 | 错误显示 + 重试按钮 |
| AC5 | Esc 键 | AIPanel 打开 | 面板关闭 |
| AC6 | Esc 键 | 有未发送内容 | 确认弹窗 |
| AC7 | 响应过长 | 响应 > 500px | 滚动条正常 |
| AC8 | Ctrl+Enter | 输入框聚焦 | 触发发送 |
