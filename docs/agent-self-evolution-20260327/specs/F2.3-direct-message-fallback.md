# Spec: F2.3 — 直接消息通知降级

## 功能概述
当 Feishu API 超限时，降级为直接发送消息（不依赖群组通知），确保关键通知仍可送达。

## 实现方案

在心跳脚本中增加 fallback 分支:

```bash
# 方案: 当群组消息失败时，尝试直接消息
if ! openclaw message send --channel feishu --target "oc_8bf15f..."; then
  echo "⚠️ 群组消息失败，尝试直接消息..."
  # 通过 sessions_send 直接通知 coord agent
  openclaw sessions send \
    --session coord-main \
    --message "PM 心跳: $(date) — 无待处理任务"
fi
```
