限界上下文 (9)
项目管理核心域
负责项目的全生命周期管理，包括项目创建、启动、执行、监控和收尾。提供项目模板、项目的状态跟踪和整体进度概览。
任务管理核心域
负责项目任务的具体管理，包括任务创建、分配、状态流转和优先级管理。协调任务间的依赖关系和执行顺序。
团队管理支撑域
管理项目团队成员信息、角色权限和部门组织架构。负责团队成员的邀请、移除和权限分配。
计划排期支撑域
管理项目的时间线和进度安排，包括里程碑设置、甘特图展示和截止日期提醒。
资源管理支撑域
管理项目资源分配和利用率，包括人力、设备的分配与调度，优化资源使用效率。
文档管理支撑域
统一管理项目相关的文档和附件，提供版本控制、协作编辑和分类检索功能。
通知中心通用域
统一处理系统内的各种通知消息，包括站内信、邮件通知和消息推送，支持个性化订阅配置。
用户认证通用域
提供用户身份认证和授权服务，包括登录登出、密码管理和单点登录功能。
审计日志通用域
记录系统操作日志，支持操作追溯、合规审计和异常分析，满足监管合规要求。
