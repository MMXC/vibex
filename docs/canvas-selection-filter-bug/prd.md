## 2. Bug 描述

### 问题
**选择卡片后点击继续，请求体未只传勾选数据**

### 预期行为
用户勾选特定卡片后点击「继续」，系统应只发送用户勾选的卡片 ID 到后端。

### 实际行为
系统发送了所有卡片数据（包括未勾选的），或筛选逻辑不正确。

### 影响范围
- Canvas 卡片选择功能
- 后续处理流程（依赖正确筛选的数据）
