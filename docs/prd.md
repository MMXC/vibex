## 2. 问题与修复方案

### 2.1 P0: 首页显示默认模板

| 问题 | 描述 |
|------|------|
| 当前状态 | 首页 (/) 显示 "Create Next App" |
| 影响 | 用户无法从首页进入 |

**修复方案**:

| 步骤 | 操作 | 验收标准 |
|------|------|----------|
| F2.1.1 | 修复 / 路由配置 | 首页显示正确内容 |
| F2.1.2 | 修复 metadata | 标题显示 "VibeX" |
| F2.1.3 | 部署验证 | 访问 / 显示正确页面 |

### 2.2 P0: 登录页无注册入口

| 问题 | 描述 |
|------|------|
| 当前状态 | "还没有账号？" 文本后无注册链接 |
| 影响 | 新用户无法注册 |

**修复方案**:

| 步骤 | 操作 | 验收标准 |
|------|------|----------|
| F2.2.1 | 添加注册链接 | "还没有账号？" 后显示 "注册" 链接 |
| F2.2.2 | 链接跳转 | 点击后跳转到注册表单或 /auth?mode=register |
| F2.2.3 | 表单切换 | 注册/登录表单可切换 |

### 2.3 P1: 演示页无引导注册

| 问题 | 描述 |
|------|------|
| 当前状态 | /chat/ 页面无引导注册提示 |
| 影响 | 用户不知道可以保存结果 |

**修复方案**:

| 步骤 | 操作 | 验收标准 |
|------|------|----------|
| F2.3.1 | 添加注册引导 | 页面显示 "登录保存对话" 提示 |
| F2.3.2 | 登录入口 | 提供登录/注册入口 |

## 6. 测试用例

### TC-001: 落地页入口
```typescript
test('should navigate to auth from landing', async ({ page }) => {
  await page.goto('https://vibex-app.pages.dev/landing/');
  await page.click('text=开始使用');
  await expect(page).toHaveURL(/auth/);
});
```

### TC-002: 注册入口显示
```typescript
test('should show register link', async ({ page }) => {
  await page.goto('https://vibex-app.pages.dev/auth/');
  await expect(page.locator('text=还没有账号？')).toBeVisible();
  await expect(page.locator('text=注册')).toBeVisible();
});
```

### TC-003: 注册链接跳转
```typescript
test('should navigate to register', async ({ page }) => {
  await page.goto('https://vibex-app.pages.dev/auth/');
  await page.click('text=注册');
  await expect(page).toHaveURL(/register/);
});
```
