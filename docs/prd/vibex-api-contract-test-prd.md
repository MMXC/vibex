## 3. 失败条件案例

| 错误类型 | 场景 | 触发条件 |
|----------|------|----------|
| SCHEMA_MISMATCH | 响应不符合 Schema | API 返回字段缺失/类型错误 |
| BREAKING_CHANGE | 破坏性变更 | 删除/重命名端点或字段 |
| TYPE_GEN_FAILED | 类型生成失败 | OpenAPI 文件语法错误 |
| CONTRACT_VERSION_MISMATCH | 契约版本不匹配 | 前后端使用不同版本契约 |
| ENDPOINT_NOT_FOUND | 端点未定义 | 访问未定义端点 |
| REQUEST_VALIDATION_FAILED | 请求验证失败 | 请求参数不符合 Schema |
| RESPONSE_VALIDATION_FAILED | 响应验证失败 | 响应数据不符合 Schema |
| OPENAPI_PARSE_ERROR | OpenAPI 解析错误 | YAML/JSON 格式错误 |

## 4. 验收标准

### P0 功能

| 验收项 | 测试方法 |
|--------|----------|
| OpenAPI 生成成功 | 运行生成命令，检查文件输出 |
| 端点删除检测 | 删除端点，运行 CI 检查 |
| 必填字段检测 | 添加必填字段，运行 CI 检查 |
| 请求验证失败测试 | 发送无效请求，检查返回 400 |
| 响应字段缺失测试 | Mock 缺失字段响应，检查测试失败 |
| 响应类型错误测试 | Mock 错误类型响应，检查测试失败 |
