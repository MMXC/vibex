## 4. 集成方案

### 4.1 首页集成流程

```
首页 (/)
    │
    ▼
显示需求录入表单
    │
    ▼
用户输入 → 点击"生成"
    │
    ▼
检查登录态
    │
    ├─ 已登录 ──▶ 跳转到 /confirm/context
    │
    └─ 未登录 ──▶ 弹出 LoginDrawer
                      │
                      ▼
                用户登录成功
                      │
                      ▼
                关闭 LoginDrawer
                      │
                      ▼
                跳转到 /confirm/context
```

### 4.2 Dashboard 布局变化

**当前**:
```
┌────────┬─────────────────────┐
│ Sidebar│     Main Content   │
└────────┴─────────────────────┘
```

**目标**:
```
┌─────────────────────────────┐
│      Navigation (顶部)       │
├─────────────────────────────┤
│                             │
│       Main Content          │
│                             │
└─────────────────────────────┘
```

## 7. 失败场景处理

| 场景 | 处理策略 |
|------|----------|
| 组件导入失败 | 降级显示基础版本 |
| 登录状态丢失 | 弹出 LoginDrawer |
| 登录被中断 | 保留输入内容 |
| 重定向循环 | 重置状态，跳转首页 |
