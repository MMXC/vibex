# Specs: E1 Proposal Template

## Template: docs/templates/proposal-submission-template.md

### Fields
| Field | Type | Required | Description |
|-------|------|---------|-------------|
| 背景 | Text | Yes | 描述项目上下文 |
| 目标 | List | Yes | 具体可测量的目标 |
| 约束 | List | Yes | 资源/时间/技术限制 |
| 成功指标 | List | Yes | 可量化的验收标准 |
| 影响范围 | Text | Yes | 涉及的功能/模块 |
| 时间线 | Date | Yes | 预期完成日期 |
| 提案者 | Text | Yes | 提交人名称 |

### Verification Checklist
- [ ] 所有 7 个字段存在
- [ ] 每个字段有灰色注释指引
- [ ] proposal ID 格式说明: `proposal-YYYYMMDD-NNN`
- [ ] sla-deadline 字段说明
- [ ] 可通过 `cp` 命令正确复制到目标目录
