## E2.2 SLA 机制规格

### SLA 计时

- **起点**: Coord 执行 `task claim` 分配给 Analyst 的时刻
- **终点**: Analyst 执行 `task update ... done` 的时刻
- **时限**: 24 小时（含工作日和节假日）
- **超时**: SLA 截止时间 + 1h 未完成，自动通知 Coord

### SLA 状态字段

```markdown
> **SLA Deadline**: 2026-04-15 12:33 (+24h)
> **SLA Remaining**: 18h 23m
```

### SLA 超时通知

```bash
# 超时消息模板
[Analyst] ⚠️ SLA 超时警报
提案: proposal-20260414-001
分配时间: 2026-04-14 12:33
SLA 截止: 2026-04-15 12:33
已超时: 1h 23m
负责人: Analyst Agent
请 Coord 确认是否继续或升级
```
