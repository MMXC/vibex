# Spec: F11.2 — 401 错误 UI 层差异化展示

## 概述

`useVersionHistory` hook 捕获 API 错误后仅打印日志，未将错误状态暴露给 UI 组件。导致用户看到"加载失败"但不知道是未登录还是网络问题。

## 根因

- `canvasApi.handleResponseError` 已正确抛出含用户友好消息的 Error（`'登录已过期，请重新登录'`）
- `useVersionHistory.loadSnapshots()` 的 catch block 只调用 `canvasLogger.default.error()`，未设置 UI 可读状态
- `VersionHistoryPanel` 有 error banner UI，但读取的是组件内部 state，而非 hook 暴露的 error

## 改动文件

| 文件 | 改动类型 |
|------|---------|
| `vibex-frontend/src/hooks/canvas/useVersionHistory.ts` | 修改 |
| `vibex-frontend/src/components/canvas/features/VersionHistoryPanel.tsx` | 修改 |

## 规格 1: `useVersionHistory.ts` 改动

### 1.1 接口扩展

```ts
interface UseVersionHistoryReturn {
  // ...existing fields
  error: string | null;  // ← 新增
}
```

### 1.2 State 添加

```ts
const [error, setError] = useState<string | null>(null);
```

### 1.3 `loadSnapshots` catch block

```ts
} catch (err) {
  canvasLogger.default.error('[useVersionHistory] loadSnapshots error:', err);
  const msg = err instanceof Error ? err.message : '加载失败，请重试';
  setError(msg);  // ← 新增
}
```

### 1.4 `createSnapshot` catch block

```ts
} catch (err) {
  canvasLogger.default.error('[useVersionHistory] createSnapshot error:', err);
  const msg = err instanceof Error ? err.message : '创建快照失败，请重试';
  setError(msg);  // ← 新增
}
```

### 1.5 `open()` 清除旧错误

```ts
const open = useCallback(() => {
  setIsOpen(true);
  setError(null);  // ← 新增：打开时清除旧错误
  loadSnapshots();
}, [loadSnapshots]);
```

## 规格 2: `VersionHistoryPanel.tsx` 改动

### 2.1 改为从 hook 读取 error

```tsx
// 移除 local error state for load/create（保留 restoreError）
// const [error, setError] = useState<string | null>(null);  ← 删除

// 从 hook 读取：
const error = versionHistory.error;

// 保留 restoreError 用于 restore 操作的本地错误
const [restoreError, setRestoreError] = useState<string | null>(null);
```

### 2.2 error banner 逻辑（已有，确认使用 hook error）

`VersionHistoryPanel.tsx` 已有 error banner UI（`<div role="alert">`），确认其读取 `error` 而非本地 state。

## 边界情况

| 情况 | 预期行为 |
|------|---------|
| 401（未登录）| "登录已过期，请重新登录" |
| 网络失败 | "加载失败，请重试" |
| 创建快照失败 | "创建快照失败，请重试" |
| 关闭面板重开 | error 清除，无 banner |
| 连续点击两次"历史" | 清除旧 error，显示新 error |
