## 1. 问题定义

**问题**: 流程树画布区域缺少「继续·组件树」按钮，用户无法触发组件树生成流程

**影响**: 用户在完成流程树编辑后，无法继续下一步（组件树生成）

---

## 2. 现状分析

### 2.1 问题定位

| 层级 | 可能的文件/位置 | 说明 |
|------|-----------------|------|
| 画布主文件 | `src/app/canvas/page.tsx` | 画布布局和按钮区域 |
| 流程树组件 | `src/components/canvas/flow-tree/` | 流程树渲染区域 |
| 按钮组件 | `src/components/canvas/buttons/` 或内联 | 按钮定义 |
| 状态管理 | `useDDDStore` | 按钮可用状态 |
| API 调用 | `canvasApi.ts` 或类似 | 组件树请求 |

### 2.2 预期行为

1. 用户在流程树区域完成编辑
2. 点击「继续·组件树」按钮
3. 系统发送流程数据到后端
4. 后端返回组件树卡片数据
5. 页面渲染组件树卡片
