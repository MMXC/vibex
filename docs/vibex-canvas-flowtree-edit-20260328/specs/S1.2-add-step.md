## 6.1 Store 层修改

### 6.1.1 新增 action: `addStepToFlow`

**文件**: `src/lib/canvas/canvasStore.ts`

```typescript
addStepToFlow: (flowNodeId: string, data?: Partial<FlowStep>) => void;
```

**实现**:
```typescript
addStepToFlow: (flowNodeId, data) => {
  const { flowNodes } = get();
  const flowNode = flowNodes.find((n) => n.nodeId === flowNodeId);
  if (!flowNode) return;

  const newStep: FlowStep = {
    stepId: generateId(),
    name: data?.name ?? '',
    actor: data?.actor ?? '',
    description: data?.description ?? '',
    order: flowNode.steps.length,
    confirmed: false,
    status: 'pending',
    type: 'normal',
  };

  set((s) => ({
    flowNodes: s.flowNodes.map((n) =>
      n.nodeId === flowNodeId ? { ...n, steps: [...n.steps, newStep] } : n
    ),
  }));
},
```

**接口更新** (第 195 行附近):
```typescript
addStepToFlow: (flowNodeId: string, data?: Partial<FlowStep>) => void;
```

### 6.2.3 自动进入编辑模式

在 `BusinessFlowTree` 中维护 `editingStepId` state：

```typescript
const [editingStepId, setEditingStepId] = useState<string | null>(null);

const handleAddStep = useCallback((flowNodeId: string) => {
  addStepToFlow(flowNodeId);
  // 找到最新添加的 stepId
  const flow = useCanvasStore.getState().flowNodes.find(n => n.nodeId === flowNodeId);
  if (flow && flow.steps.length > 0) {
    const lastStep = flow.steps[flow.steps.length - 1];
    setEditingStepId(lastStep.stepId);
  }
}, [addStepToFlow]);
```

## 6.5 风险评估

| 风险 | 概率 | 影响 | 缓解 |
|------|------|------|------|
| 快速连续点击添加多个步骤 | 低 | 低 | 添加防抖（debounce）|
| 编辑状态与数据不同步 | 中 | 中 | StepRow 使用受控组件 |
