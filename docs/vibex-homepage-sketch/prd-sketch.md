## 区域规格

### 1. HEADER (8% 高度)
| 元素 | 位置 | 说明 |
|------|------|------|
| Logo + 标语 | 左侧 | `🤖 AI 驱动的应用构建平台` |
| Nav Links | 右侧 | 未登录: [免费开始] [查看演示] |
| | | 已登录: [项目管理] [个人设置] |

### 2. 主内容区 (三栏布局)

#### 2.1 SIDEBAR (15% 宽度)
- **功能**: 设计流程步骤导航
- **步骤**:
  1. 需求输入
  2. 限界上下文
  3. 领域模型
  4. 业务流程
  5. 项目创建
- **交互**: 可随时切换查看已完成步骤

#### 2.2 PREVIEW 区域 (60%+ 区域)
- **功能**: AI 生成结果实时预览
- **内容**: 三种图表 (限界上下文/领域模型/业务流程)
- **特性**: 画布可调整大小
- **INPUT 子区域**: 底部固定
  - 需求输入框 + 发送按钮
  - GitHub/Figma 导入入口
  - 诊断面板

#### 2.3 AI PANEL (15% 宽度)
- **功能**: 点击生成时展示 AI 思考过程
- **触发**: 用户点击"发送"后激活
- **内容**: 实时流式显示 AI 分析思路

### 3. FOOTER
- **内容**: Copyright 信息
- **样式**: 固定底部
