## 当前代码

```typescript
// canvasApi.ts ~line 310
return result.components.map((comp) => ({
  flowId: (comp.flowId && comp.flowId !== 'unknown') ? comp.flowId : '',
  name: comp.name,
  type: comp.type as ComponentType,
  props: {},  // ← 修改这里
  api: comp.api ?? { ... },
  nodeId: `comp-${Date.now()}-${Math.random()}`,
  confirmed: false,
  status: 'pending' as const,
  children: [],
}));
```

## 修复方案

将 `props: {}` 替换为：

```typescript
props: generateDefaultProps(comp.type, comp.name),
```

## 验收标准

```typescript
const nodes = await fetchComponentTree(mockResponse);

  // props 不再为空对象
  expect(node.props).not.toEqual({});

  expect(node.props.title).toBe(node.name);

```
