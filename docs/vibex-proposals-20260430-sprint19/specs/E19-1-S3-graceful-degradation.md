# Spec: E19-1-S3 — 优雅降级

**Story**: E19-1-S3
**验收标准**: AS3.1–AS3.4
**工时**: 0.5d

---

## 概述

确保 Design Review 在网络异常、服务器错误、加载中、空数据四种状态下均有友好反馈，不出现白屏、不显示假数据。

---

## 状态机

```
idle → loading → success(result)
                → error(errorMsg)
                
idle（初始/关闭后）→ 显示空状态引导文案
loading → 显示骨架屏或加载动画
error → 显示友好文案 + 重试按钮
success → 显示真实评审结果
```

---

## UI 文案规范

| 状态 | 文案 | 样式 |
|------|------|------|
| 加载中 | "正在分析设计，请稍候..." | 骨架屏 + loading spinner |
| 网络错误 | "网络连接异常，请检查网络后重试" | 警告图标 + 重试按钮 |
| 服务端错误 | "设计评审暂时不可用，请稍后再试" | 错误图标 + 重试按钮 |
| 无数据/空状态 | "暂无评审结果，按 Ctrl+Shift+R 触发评审" | 提示图标 + 引导文案 |

---
