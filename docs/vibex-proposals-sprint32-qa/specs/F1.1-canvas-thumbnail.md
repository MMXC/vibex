# F1.1 Canvas 缩略图导航 — 四态规格

**Epic**: F1.1 Canvas 缩略图导航
**涉及文件**: `vibex-fronted/src/components/dds/canvas/CanvasThumbnail.tsx` + `CanvasThumbnail.module.css`
**集成页面**: CanvasPage（`vibex-fronted/src/components/canvas/CanvasPage.tsx`）

---

## 状态机概览

```
                    ┌─────────────────────┐
                    │  HIDE (条件不满足)  │
                    │  节点数 < threshold  │
                    │  → return null      │
                    └──────────┬──────────┘
                               │ 节点数 ≥ threshold
                               ▼
┌──────────────────┐    ┌──────────────────┐    ┌──────────────────┐
│  RENDERING (加载) │───▶│  IDLE (理想态)   │───▶│  INTERACTING     │
│  渲染中，无指示器 │    │  缩略图+指示器    │    │  用户点击/拖拽中  │
└──────────────────┘    └──────────────────┘    └──────────────────┘
                               ▲
                               │ 节点数跌破 threshold
                               │
                         ┌─────┴─────┐
                         │  HIDE    │
                         └──────────┘
```

---

## 态 1：理想态（IDLE）

**触发条件**: 节点数 ≥ threshold（默认 50），组件已挂载

**视觉规范**:
- 容器固定在画布右侧 panel，宽度 160px，内边距 6px
- 背景: `var(--color-surface-subtle, #f9fafb)`
- 边框: `1px solid var(--color-border, #e5e7eb)`
- 圆角: 6px

**SVG 规范**:
- 尺寸: 148×100（内边距 12px 留白）
- 节点矩形: `fill: var(--color-node-fill, #e0e7ff)` + `stroke: var(--color-node-stroke, #6366f1)` + `stroke-width: 0.5` + `opacity: 0.7`
- 视口指示器: `fill: rgba(99, 102, 241, 0.15)` + `stroke: var(--color-primary, #6366f1)` + `stroke-width: 1.5`
- 指示器最小尺寸: 4×4px（防止极小视口时不可见）

**交互规范**:
- 鼠标悬停: cursor: pointer
- focus-visible: `outline: 2px solid var(--color-focus-ring, #6366f1)` + `outline-offset: 2px`
- 键盘: Enter/Space 触发跳转（无障碍）

**无障碍属性**:
- 容器: `role="img"` + `aria-label="画布缩略图"`
- SVG: `role="img"` + `aria-label="点击跳转到对应区域"`
- data-testid: `data-testid="canvas-thumbnail"`

**引导文案**: 无（理想态无文案，纯视觉导航）

---

## 态 2：空状态（HIDE）

**触发条件**: 节点数 < threshold（默认 50）

**视觉规范**:
- `return null`，组件不渲染任何 DOM

**E2E 验收断言**:
```javascript
// 节点数不足时不应渲染
expect(screen.queryByTestId('canvas-thumbnail')).not.toBeInTheDocument()
// 等价于: expect(screen.queryByTestId('canvas-thumbnail')).toBeNull()
```

**引导文案**: 无（不应出现空状态文案，避免误导用户"缩略图坏了"）

---

## 态 3：加载态（RENDERING）

**触发条件**: 节点数刚达到 threshold，指示器尚未计算完毕

**视觉规范**:
- 同理想态，但指示器 rect 尚未出现（`indicatorRect === null`）

**时序**:
1. `getNodes()` 返回 ≥ threshold 个节点 → 进入 RENDERING
2. 调用 `updateIndicator()` → 同步计算 worldToThumbnail
3. `setIndicatorRect(...)` → 切换到 IDLE
4. 指示器出现（~1 帧内完成，用户无感知）

**E2E 验收断言**:
```javascript
// 加载态不应出现空白/崩溃
expect(screen.getByTestId('canvas-thumbnail')).toBeInTheDocument()
// SVG 应渲染节点矩形（即使指示器尚未出现）
expect(screen.getByTestId('canvas-thumbnail').querySelectorAll('rect').length).toBeGreaterThan(0)
```

---

## 态 4：错误态（ERROR）

**触发条件**: `getNodesBounds()` 返回 `null`（节点存在但 bounds 无法计算）

**视觉规范**:
- 同理想态（节点矩形 + 指示器不渲染，因为 `nodesRect === null`）
- 组件提前返回 `null`，静默降级

**E2E 验收断言**:
```javascript
// 边界场景：节点存在但 bounds 为 null → 不渲染缩略图
expect(screen.queryByTestId('canvas-thumbnail')).not.toBeInTheDocument()
// 不应崩溃，不应显示错误文案
```

**兜底机制**: `if (!nodesRect) return null` — 静默降级，不打断用户工作流

## CSS Token 清单

| 属性 | Token | 备选值 |
|------|-------|--------|
| 容器背景 | `--color-surface-subtle` | `#f9fafb` |
| 边框色 | `--color-border` | `#e5e7eb` |
| 圆角 | （固定值） | `6px` |
| 节点填充 | `--color-node-fill` | `#e0e7ff` |
| 节点描边 | `--color-node-stroke` | `#6366f1` |
| 指示器填充 | （rgba 合成） | `rgba(99, 102, 241, 0.15)` |
| 指示器描边 | `--color-primary` | `#6366f1` |
| Focus Ring | `--color-focus-ring` | `#6366f1` |
| 内边距（容器） | （固定值） | `6px` |
| SVG 内边距 | （固定值） | `8px` |

> ⚠️ 本规格中所有颜色/间距均为 CSS Token 或固定值，无硬编码十六进制值。实际代码应使用 `var(--*)` 语法引用上述 Token。

## E2E 完整验收脚本

```javascript
describe('F1.1 Canvas 缩略图导航', () => {
  it('节点数 < 50 时不渲染缩略图（空状态）', () => {
    renderCanvasWithNodes(49)
    expect(screen.queryByTestId('canvas-thumbnail')).toBeNull()
  })

  it('节点数 ≥ 50 时渲染缩略图（理想态）', () => {
    renderCanvasWithNodes(51)
    expect(screen.getByTestId('canvas-thumbnail')).toBeVisible()
    expect(screen.getByTestId('canvas-thumbnail')).toHaveAttribute('role', 'img')
  })

  it('指示器 rect 存在且可见（理想态）', () => {
    renderCanvasWithNodes(51)
    const svg = screen.getByTestId('canvas-thumbnail').querySelector('svg')
    const indicator = svg.querySelector('[class*="indicator"]')
    expect(indicator).toBeVisible()
  })

  it('点击缩略图触发 setViewport（交互态）', async () => {
    const setViewport = vi.fn()
    renderCanvasWithNodes(51, { setViewport })
    await userEvent.click(screen.getByTestId('canvas-thumbnail').querySelector('svg'))
    expect(setViewport).toHaveBeenCalled()
  })

  it('节点数跌破 threshold 后缩略图消失（空状态）', async () => {
    renderCanvasWithNodes(51)
    expect(screen.getByTestId('canvas-thumbnail')).toBeVisible()
    // 模拟节点删除
    removeNodes(49)
    await waitFor(() => {
      expect(screen.queryByTestId('canvas-thumbnail')).toBeNull()
    })
  })

  it('无障碍：aria-label 和 role 正确', () => {
    renderCanvasWithNodes(51)
    const container = screen.getByTestId('canvas-thumbnail')
    expect(container).toHaveAttribute('role', 'img')
    expect(container).toHaveAttribute('aria-label', '画布缩略图')
  })

  it('无障碍：SVG 有 img role 和描述性 aria-label', () => {
    renderCanvasWithNodes(51)
    const svg = screen.getByTestId('canvas-thumbnail').querySelector('svg')
    expect(svg).toHaveAttribute('role', 'img')
    expect(svg).toHaveAttribute('aria-label', '点击跳转到对应区域')
  })
})
```
