# Spec: F1.1 Canvas 缩略图导航

## 概述

Canvas 右侧缩略图导航面板，显示整个画布的缩略图和当前视口位置，用户点击任意位置可跳转到对应区域。

解决"300+ 节点画布用户找不到节点"的导航痛点。

## 技术方案

### 组件
- 新建 `vibex-fronted/src/components/dds/CanvasThumbnail.tsx`
- 集成到 `CanvasPage.tsx` 右侧面板

### 实现
- 读取 ReactFlow `useReactFlow()` 获取当前 viewport（x, y, zoom）
- 根据 viewport 计算当前视口在缩略图中的高亮矩形（x/y/width/height 按比例换算）
- 用户点击缩略图 → 调用 `setViewport({ x, y, zoom })` 跳转
- 缩略图渲染：使用 `html2canvas` 或 Canvas2D 将实际 Canvas 内容缩放绘制

### 显示条件
- 节点数量 >= 50 时显示缩略图
- 节点数量 < 50 时隐藏

## 数据结构

```typescript
interface ThumbnailState {
  viewportX: number;
  viewportY: number;
  viewportWidth: number;
  viewportHeight: number;
  scale: number; // 缩略图 scale
}
```

## 交互

1. 拖拽实际 Canvas → 缩略图视口高亮同步更新
2. 拖拽缩略图视口高亮 → 实际 Canvas viewport 跳转
3. 滚轮缩放 → 缩略图视口大小同步变化
