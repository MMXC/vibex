# Spec: F1.3 离线写入队列

## 概述

当前 PWA 离线模式是"只读离线"（GET 可缓存，非 GET 离线直接 503）。Sprint 32 升级为"读写离线"：非 GET 请求在离线时写入 IndexedDB 队列，联网后自动重放。

## 技术方案

### 文件
- 修改 `vibex-fronted/public/sw.js`
- 新建 `vibex-fronted/src/lib/offline-queue.ts`

### IndexedDB Schema

```typescript
// database: vibex-offline, store: offline-queue
interface QueuedRequest {
  id: string; // uuid
  url: string;
  method: 'POST' | 'PUT' | 'DELETE';
  headers: Record<string, string>;
  body: string | null;
  timestamp: number;
  retryCount: number;
}
```

### SW 拦截逻辑

```javascript
// sw.js
self.addEventListener('fetch', (event) => {
  if (event.request.method === 'GET') return; // 保持现有缓存逻辑

  if (!navigator.onLine) {
    event.respondWith(queueRequest(event.request));
    return;
  }

  // 在线：正常请求
});
```

### 同步逻辑

```typescript
// offline-queue.ts
export async function replayQueue() {
  const queue = await idb.getAll('offline-queue');
  for (const req of queue.sort((a, b) => a.timestamp - b.timestamp)) {
    try {
      await fetch(req.url, { method: req.method, body: req.body, headers: req.headers });
      await idb.delete('offline-queue', req.id);
    } catch (e) {
      if (req.retryCount >= 3) {
        await idb.delete('offline-queue', req.id);
        notifyUser(`同步失败: ${req.method} ${req.url}`);
      } else {
        await idb.put('offline-queue', { ...req, retryCount: req.retryCount + 1 });
      }
    }
  }
}
```

### Feature Flag

`VITE_ENABLE_OFFLINE_QUEUE=true/false` — 关闭时降级为只读离线。

## 验收标准

```
const req = new Request('/api/projects/1', { method: 'PUT', body: '{"name":"x"}' });
await sw.handleRequest(req); // offline
const queued = await idb.getAll('offline-queue');
expect(queued.length).toBe(1);

await goOnline();
expect(fetch).toHaveBeenCalledWith(expect.objectContaining({ method: 'PUT' }));
expect(await idb.getAll('offline-queue')).toHaveLength(0);
```
