# Spec: F2.1 Vitest Snapshot 测试

## 概述

为 `ChapterPanel` 和 `DDSCanvasStore` 添加 snapshot 测试，防止组件渲染结果意外退化。

## 技术方案

### 文件
- 新建 `vibex-fronted/src/components/dds/__tests__/ChapterPanel.test.tsx`
- 新建 `vibex-fronted/src/stores/dds/__tests__/DDSCanvasStore.test.ts`

### ChapterPanel Snapshot

```typescript
import { render } from '@testing-library/react';
import { ChapterPanel } from '../ChapterPanel';

describe('ChapterPanel snapshot', () => {
  it('renders without crashing', () => {
    const { container } = render(<ChapterPanel />);
    expect(container).toMatchSnapshot();
  });

  it('renders with cards', () => {
    const { container } = render(<ChapterPanel cards={mockCards} />);
    expect(container).toMatchSnapshot();
  });
});
```

### DDSCanvasStore Snapshot

```typescript
import { DDSCanvasStore } from '../DDSCanvasStore';

describe('DDSCanvasStore snapshot', () => {
  it('initial state snapshot', () => {
    const store = new DDSCanvasStore();
    expect(store).toMatchSnapshot();
  });

  it('state after selectCard', () => {
    const store = new DDSCanvasStore();
    store.selectCard('card-1');
    expect(store).toMatchSnapshot();
  });
});
```
