# Spec: F2.2 Playwright 视觉回归

## 概述

Playwright screenshot diff 防止 UI 意外破坏。核心页面截图对比，CI 报告视觉 diff。

## 技术方案

### 文件
- 新建 `vibex-fronted/tests/e2e/visual-regression.spec.ts`
- 新建 `.github/workflows/visual-regression.yml`

### visual-regression.spec.ts

```typescript
import { test, expect } from '@playwright/test';

test.describe('Visual Regression', () => {
  test('CanvasPage visual snapshot', async ({ page }) => {
    await page.goto('/canvas/test-project');
    await page.waitForLoadState('networkidle');
    await expect(page).toHaveScreenshot('canvas-page.png', { maxDiffPixels: 100 });
  });

  test('Dashboard visual snapshot', async ({ page }) => {
    await page.goto('/dashboard');
    await page.waitForLoadState('networkidle');
    await expect(page).toHaveScreenshot('dashboard.png', { maxDiffPixels: 100 });
  });
});
```

### CI Workflow

```yaml
# .github/workflows/visual-regression.yml
on: [pull_request]
jobs:
  visual-diff:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: npm ci
      - run: npx playwright install
      - run: npm run test:e2e:visual:ci
      - uses: actions/upload-artifact@v4
        if: failure()
        with:
          name: visual-diff
          path: test-results/
```

### 首次运行

首次运行生成 baseline screenshots（签入 Git），后续运行对比。

## 验收标准

```
await page.goto('/canvas/test-project');
await expect(page).toHaveScreenshot('canvas-page.png');
git status '**/__snapshots__/**'
expect(exitCode).toBe(0);
```
