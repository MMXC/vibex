## 1. 理想态

用户首次进入原型编辑器，画布完全空白。

### UI 描述

- 画布区域：居中显示引导内容
- 引导文案：`"从左侧拖拽组件到画布，开始创建你的原型"`
- 引导箭头：指向左侧组件面板
- 组件面板：默认展开，显示 10 个默认组件缩略图

### 理想态示意图

```
┌──────────────────────────────────────────────────────┐
│  [组件库]  [路由树]                                    │
├────────────┬─────────────────────────────────────────┤
│            │                                         │
│  组件面板   │         ┌─────────────────────────┐   │
│  ────────  │         │                         │   │
│  [Button]  │         │    拖拽组件到画布        │   │
│  [Input]   │         │         ↓               │   │
│  [Card]    │         │    [组件面板]           │   │
│  [Table]   │         │                         │   │
│  [Form]    │         └─────────────────────────┘   │
│  ...       │                                         │
│            │                                         │
└────────────┴─────────────────────────────────────────┘
```

## 2. 空状态 — 组件面板为空

### 触发条件

- 组件面板数据加载失败
- 默认组件库为空

### UI 描述

- 组件面板显示：插图（空白文件夹）+ 文案 `"组件库为空"`
- 副文案：`"请联系管理员添加组件"`
- 按钮：`"刷新"` — 点击重新加载

### 禁止事项

- ❌ 不留白（必须有插图+文案）
- ❌ 不显示加载中（这是错误状态，不是加载状态）
- ❌ 不显示 error stack

---

## 3. 空状态 — 路由列表为空

### 触发条件

- 项目中还没有创建任何页面
- 所有页面被删除

### UI 描述

- 左侧抽屉路由列表区域：
  - 插图（空白地图）+ 文案 `"还没有添加页面"`
  - 副文案：`"请先在画布上创建页面"`
  - 引导箭头：指向画布区域

### 禁止事项

- ❌ 不留白（必须有插图+文案）
- ❌ 不显示数字 `0`
- ❌ 不说"没有路由"（说"没有页面"更直观）

---

## 4. 空状态 — 画布为空（已有节点后清空）

### 触发条件

- 用户删除了画布上的所有节点
- 页面内容被清空

### UI 描述

- 画布显示：
  - 半透明遮罩 + 文案 `"画布已清空"`
  - 按钮：`"撤销"` — 恢复最近删除的节点
  - 副文案：`"或从左侧重新拖入组件"`

### 禁止事项

- ❌ 不强制恢复（用户可能故意清空）
- ❌ 不自动恢复（需用户主动点击）

---

## 5. 空状态 — Mock数据为空

### 触发条件

- 用户在 Mock数据 Tab 中未填写任何数据
- 组件未绑定 Mock 数据

### UI 描述

- Mock数据 Tab 显示：
  - JSON 示例模板（可折叠）
  - 文案：`"暂无 Mock 数据"`
  - 副文案：`"填写上方 JSON 或点击下方示例填充"`
  - 快捷按钮：`"填充示例数据"` — 填入预设的 Mock JSON

### 禁止事项

- ❌ 不阻止用户保存空数据（允许无 Mock 数据的组件存在）
- ❌ 不自动填充（用户可能故意为空）

---

## 6. 空状态 — 导出结果为空

### 触发条件

- 用户点击导出，但画布没有任何内容
- 导出前校验发现无有效数据

### UI 描述

- Toast 提示：`"画布为空，无法导出"`
- 副文案：`"请先添加组件后再试"`

### 禁止事项

- ❌ 不生成空文件
- ❌ 不静默失败

---

## 验收标准

```
expect(组件面板为空时显示插图+文案).toBe(true)
expect(路由列表为空时显示插图+文案).toBe(true)
expect(画布清空时显示遮罩+撤销按钮).toBe(true)
expect(Mock数据为空时显示示例模板+填充按钮).toBe(true)
expect(导出空画布时Toast提示).toBe(true)
expect(所有空状态都有引导插图).toBe(true)
expect(所有空状态文案不超过2行).toBe(true)
```
