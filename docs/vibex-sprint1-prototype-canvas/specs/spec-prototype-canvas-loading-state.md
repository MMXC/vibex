# Spec — 原型画布加载态规格

**文件**: spec-prototype-canvas-loading-state.md
**状态**: v1.0
**日期**: 2026-04-17

---

## 1. 加载态原则

- **必须使用骨架屏**，禁止使用转圈（Spinner）
- 骨架屏应模拟实际内容的布局
- 加载完成后骨架屏淡出，内容淡入（300ms 过渡）

---

## 2. 加载态 — 画布初始化

### 触发条件

- 用户进入原型编辑器
- 加载已有项目数据

### UI 描述

- 画布区域：显示骨架屏网格（模拟节点分布）
- 组件面板：左侧显示 10 个组件的骨架占位
- 顶部：进度条（确定进度）或骨架屏（不确定进度）
- 预计等待时间：若 > 2s，显示 `"正在加载原型..."`

### 骨架屏示意

```

│  [组件库]  [路由树]                                    │
├────────────┬─────────────────────────────────────────┤
│            │  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓   │
│  ▓▓▓▓▓▓    │  ▓▓▓        ▓▓▓        ▓▓▓▓▓▓▓▓▓▓    │
│  ▓▓▓▓▓▓    │  ▓▓▓        ▓▓▓        ▓▓▓▓▓▓▓▓▓▓    │
│  ▓▓▓▓▓▓    │  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓   │
│  ▓▓▓▓▓▓    │                                         │
│  ▓▓▓▓▓▓    │                                         │
│  ▓▓▓▓▓▓    │                                         │
└────────────┴─────────────────────────────────────────┘
```

### 禁止事项

- ❌ 禁止使用 `<Loading />` 或 `<Spinner />`
- ❌ 禁止只显示 loading 文字
- ❌ 禁止白屏等待

---

## 3. 加载态 — 组件拖入中

### 触发条件

- 用户正在拖拽组件，但尚未释放

### UI 描述

- 画布：显示拖拽预览（半透明节点）
- 组件面板：被拖拽的组件显示为半透明（原位置保留虚影）
- 光标：保持拖拽状态

### 禁止事项

- ❌ 不显示加载指示器（拖拽是即时操作）
- ❌ 不阻止用户释放

---

## 4. 加载态 — 属性面板加载

### 触发条件

- 双击节点，打开属性面板
- 属性面板数据较大，需要加载

### UI 描述

- 右侧属性面板：显示骨架屏（Tab 区域 + 表单骨架）
- 骨架屏高度与实际内容一致，避免跳跃

### 禁止事项

- ❌ 禁止属性面板闪烁
- ❌ 禁止内容高度跳动

---

## 5. 加载态 — Mock数据加载

### 触发条件

- 组件绑定了外部 Mock 数据源（方案 B/C，Phase 2）
- 本期 MVP 使用内嵌 Mock（方案 A），此场景暂不触发

### UI 描述（Phase 2 预留）

- Mock数据 Tab 显示：骨架屏行（模拟 JSON 行）
- 预计时间 > 2s：显示 `"正在加载 Mock 数据..."`

### 禁止事项

- ❌ 本期 MVP 不实现此加载态（内嵌 Mock 无需加载）

---

## 6. 加载态 — 路由列表加载

### 触发条件

- 打开路由列表抽屉
- 页面数据较多，需要加载

### UI 描述

- 抽屉内容：显示 5 条路由列表骨架行
- 每行骨架：`[○] ▓▓▓▓▓▓▓▓▓▓▓▓`（图标 + 文字）

### 禁止事项

- ❌ 不显示 Spinner
- ❌ 不在抽屉外部显示加载状态

---

## 7. 加载态 — JSON导出中

### 触发条件

- 用户点击导出，生成 JSON 文件

### UI 描述

- 导出按钮：变为 loading 状态（文字变为 `"导出中..."`）
- 按钮禁用，不可重复点击
- 完成后：自动触发下载，不显示额外 UI

### 禁止事项

- ❌ 不显示模态框（导出是快速操作）
- ❌ 不阻止用户继续操作其他区域

---

## 验收标准

```
expect(画布初始化显示骨架屏).toBe(true)
expect(骨架屏模拟实际节点布局).toBe(true)
expect(组件面板加载显示骨架占位).toBe(true)
expect(属性面板加载无高度跳动).toBe(true)
expect(路由列表加载显示骨架行).toBe(true)
expect(导出中按钮显示loading文字).toBe(true)
expect(所有骨架屏使用灰色块).toBe(true)
expect(加载完成有淡入过渡).toBe(true)
```
