## 3. 失败测试详情

### 🔴 DDSScrollContainer — 8 个测试全部失败

**错误**: `TypeError: Cannot read properties of undefined (reading 'loading')`

```
src/components/dds/canvas/DDSScrollContainer.tsx:206
const data = chapters[chapter];  // undefined!
```

**根因**: Store fixture key vs CHAPTER_ORDER key 不一致

| 位置 | Key | 值 |
|------|-----|------|
| Store fixture (test) | `businessRules` (camelCase) | `{ type: 'business-rules', ... }` |
| CHAPTER_ORDER (component) | `'business-rules'` (kebab-case) | - |

**访问路径**:
```typescript
// Component (DDSScrollContainer.tsx:184)
// data = chapters[chapter] where chapter = 'business-rules'
const data = chapters['business-rules']; // undefined! (store has 'businessRules')
```

**修复建议**:
选项 A: 统一为 kebab-case key
```typescript
// Store fixture
{ 'business-rules': { type: 'business-rules', cards: [], edges: [], ... } }
// Component CHAPTER_ORDER: ['requirement', 'context', 'flow', 'api', 'business-rules']
```
选项 B: 统一为 camelCase key
```typescript
// CHAPTER_ORDER: ['requirement', 'context', 'flow', 'api', 'businessRules']
// Store fixture: { businessRules: { type: 'business-rules', ... } }
```
