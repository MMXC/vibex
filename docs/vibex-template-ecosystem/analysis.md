基于 v0.dev 竞品分析，v0.dev 拥有 **13 个模板分类**，热门模板使用量达 **18.5K+**。建议 VibeX 预置 **10 个行业模板**，覆盖电商、教育、社交、企业等核心场景，预期提升新用户转化率 **40%**。

## 4. 技术实现方案

### 4.1 模板数据结构

```typescript
interface Template {
  id: string;
  name: string;
  industry: string;
  complexity: 'low' | 'medium' | 'high';
  description: string;
  requirement: string; // 预置需求描述
  boundedContexts: BoundedContext[];
  previewImage: string;
  useCount: number;
}

interface BoundedContext {
  name: string;
  type: 'core' | 'supporting' | 'generic';
  entities: string[];
  services: string[];
}
```

### 4.2 模板存储方案

```
src/data/templates/
├── ecommerce.json       # 电商商城
├── education.json       # 在线教育
├── corporate.json       # 企业官网
├── project-management.json
├── social-community.json
├── dashboard.json       # 数据看板
├── blog.json           # 博客系统
├── booking.json        # 预约系统
├── membership.json     # 会员系统
└── landing-page.json   # 营销落地页
```

### 4.3 模板使用流程

```
1. 用户浏览模板库 → 选择模板
2. 查看模板预览 → 点击"使用模板"
3. 预置需求自动填入输入框
4. 用户修改/确认需求 → 开始设计
```
