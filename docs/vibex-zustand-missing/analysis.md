## 根因分析

zustand 被 **reactflow** 作为传递依赖引入，但 **未显式声明**。这导致：
- `pnpm` 构建正常（提升传递依赖）
- `npm ls` 报 extraneous 警告
- `npm ci` 在严格模式下可能失败

---

## 推荐方案

```bash
cd vibex-fronted
pnpm add zustand@4.5.7
```

将 `zustand@^4.5.7` 添加到 `package.json` 的 `dependencies`。
