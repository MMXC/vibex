## 优先级合并策略（ST-3.2）

优先级从高到低：

1. **localStorage** — 用户明确选择
2. **API userPreferences.theme** — 登录用户覆盖
3. **system** — OS 偏好
4. **API theme default** — 服务器默认值
5. **'system'** — 硬编码默认值

```typescript
// src/contexts/ThemeContext.tsx
function resolveTheme(
  local: ThemeMode | null,
  api: HomepageAPIResponse | null,
  system: 'light' | 'dark'
): ThemeState {
  if (local) return { mode: local, resolved: local === 'system' ? system : local };
  if (api?.userPreferences?.theme) return { mode: api.userPreferences.theme, resolved: api.userPreferences.theme };
  if (api?.theme) return { mode: api.theme === 'system' ? 'system' : api.theme, resolved: api.theme === 'system' ? system : api.theme };
  return { mode: 'system', resolved: system };
}
```
