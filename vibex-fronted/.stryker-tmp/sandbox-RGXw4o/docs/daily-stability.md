# Daily Stability Log

Generated by: `scripts/test-stability-report.sh`

## Format

| Column | Description |
|--------|-------------|
| Date | Test run date (YYYY-MM-DD HH:MM:SS) |
| PassRate | passed/total × 100% |
| Total | Total number of tests |
| Passed | Tests that passed |
| Failed | Tests that failed (first attempt) |
| Flaky | Tests that passed after retry (indicates timing sensitivity) |

## Targets

- **PassRate**: >= 95%
- **Flaky**: <= 1 per run
- **Failed**: 0 (any failure should be investigated)
